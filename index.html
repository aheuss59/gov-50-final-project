Introduction

Coming to Cambridge this fall, I had lots of things on my mind. Class choices, new roommates, new friends, and (perhaps more importantly) coffee. As I learned soon after arriving, coffee is not hard to find in Cambridge. This project is an effort to explore where coffee locations are located in the United States and why. After exploring my data with more depth, I realized that it would be ineffective and misleading to attempt to analyze all coffee shops based on the provided data. As a result, this project focuses on the chain coffee franchises Dunkin’ Donuts and Starbucks. My primary research question was inspired by the sheer prevalence of coffee in Cambridge and asks: is there a correlation between these chain coffee locations and whether or not the city it is located in is a college town?

The Data

My original data was taken from Friendly Cities on GitHub and includes 800,000 restaurants throughout the United States. The dataset included the variables: RestaurantName, Cuisine, OpenHours, State, CNTY_GEOID, UA_NAME, MSA_NAME, MSA_GEOID, Lon, Lat, Frequency, and isChain. This dataset was originally used to find correlation between chain restaurants and a number of independent variables, so this project goes in another direction. The primary variables I focused on included RestaurantName, which provided the name of the restaurant, UA_NAME, which is short for Urban Area name and was typically formatted as the city and state where the restaurant was located, and lastly the State variable, which provided the state abbreviation for the state the restaurant was located in.

As the dataset originally stood, I did not have enough data to explore my research question. I created a new tibble labeled college_towns which includes the name of every town in the United States that has a four-year higher education institution located within it. This tibble was compiled using information provided on Campus Explorer and then used to create the variable college_town within the dataset; a 1 marks that the town has a four year higher education institution and a 0 if else. I also created my dependent variable by creating a unique ID per city, which I used to create the variable is_coffee in my data. The column is marked with a 1 if the town has a chain and a 0 if not.

The rest of my variables were created as possible confounders that could provide alternative reasoning. The next two variables I added to the dataset were based on location and are called region and division. These variables were created in reference to the regions and divisions assigned to states by the United States Census Bureau. My final variable is called is_pop and marks whether or not a city was in the top 5 most populated cities within it’s respective state (1 if yes, 0 if no). This variable was created with Census Bureau data, specifically the April 2021 population estimate.

The dependent variable in my study is whether or not a town has a chain coffee location, in this study referring to a Dunkin’ Donuts or a Starbucks. My study is cross-sectional, as the data is observational, meaning that I did not assign the value for college_town, which is my primary variable of interest. Additionally, my data was collected at a single point in time and thus does not fill the criteria for a before-after or differences in differences study.

In the table above, the intercept is 0.144, which implies that when all variables are 0 (in this case, the town is not a college town and is not one of the top 5 in it’s state based on population), 14.4% of towns have a chain coffee location. Within the region variables, the Northeast variable has the largest effect with 0.377, meaning that when the town is in the Northeast states there is a 37.7% increase in towns that have chain coffee locations, holding all else equal. The population variable has a coefficient of 0.311, which indicates that when a town is in the top 5 of it’s state for population there is an average increase in towns with chain coffee locations of 31.1%, holding all else equal. Finally, the college town variable has a coefficient of 0.400, which implies that when a town is a college town, there is an average increase of 40% in the number of towns that have a chain coffee location, holding all other variable equal. This number is lower than the initial 46.9% shown in the previous table, which implies that some of the correlation might have actually been due to one of the variables in this regression. All of the coefficients in this table have very small p-values, below the aplha level of 0.05, which implies statistical significance.
